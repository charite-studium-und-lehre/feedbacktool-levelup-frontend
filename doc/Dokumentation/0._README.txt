====== 0. README ======

=== Dies ist ein Zim Wiki. ===
* Zur formatierten Darstellung zim lokal installieren:
Ubuntu: sudo apt-get install zim
und Datei -> Neues Notizbuch öffnen
* unter Bearbeiten -> Einstellungen -> Bearbeiten "KamelCaseSchreibweise automatisch verlinken" ausschalten. Erzeugt sonst bei "CamelCase" Schreibweise Links und legt neue Seiten an. Sehr nervig. 

=== Zim Manual ===
http://zim-wiki.org/manual/Start.html

=== Versionskontrolle ===
* Im git liegen alle *.txt Dateien und "notebook.zim"
* Der ".zim" Ordner ist ein lokaler Cache und liegt nicht im Versionssystem

=== Attachments ===
(Klammersymbol obem: Externe Datei anhängen, oder Werkzeuge -> Datei anhängen)
* Attachments werden, je nach Format, als Bild oder Link ins Dokument eingefügt und automatisch in einem Ordner mit dem Namen der Textdatei gespeichert!
* Attachment Ordner kann über Werkzeuge -> Anhängverzeichnis geöffnet werden
* Wird nicht automatisch gelöscht wenn Attachment wieder entfernt wird
* Größe ändern: Rechtsklick -> Eigenschaften editiieren. Wenn man nur die Höhe oder die Breite anpasst und danach in das andere Feld klickt wird dies automatisch mit beibehaltenen Proportionen angepasst


=== Bilder ===
* Bei Einfügen -> Bild wird dieses nicht automatisch im Verzeichnis gespeichert
* ODER: Drag&Drop vom Dateimanager aus
