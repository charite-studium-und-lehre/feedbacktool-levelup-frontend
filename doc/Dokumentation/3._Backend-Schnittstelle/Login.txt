====== Login ======

Login über SSO.

== Schnittstelle: ==
POST-Aufruf an Server:

/api/login


Rückgabe:
'''
{
	"vorname": <vorname>,
	"nachname": <nachname>,
	"stammdatenVorhanden": <true|false>
'''

'''
}
'''


=== Erfolg: ===
Cookie wird serverseitig vom Backend gesetzt (als "angemeldet").
success: HTTP-Code 200

=== fail: ===
HTTP-Code 404 (Nicht gefunden)
