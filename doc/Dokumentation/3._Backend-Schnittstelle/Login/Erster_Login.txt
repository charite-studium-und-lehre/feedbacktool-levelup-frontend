====== Erster Login ======

Formular:
* Matrikelnummer
* Vorname (wird nur dargestellt aus Antwort vom Server)
* Nachname (wird nur dargestellt aus Antwort vom Server)

Schnittstelle:

/api/stammdaten

== POST-Aufruf: ==

{
	"matrikelnummer" : <matrikelnummer>,
}

== Erfolg: ==
HTTP-Code 200
-> Backend: dem Benutzer wird ein Login-hash hinzugefügt, damit der Benutzer diese Seite nur einmal ausfüllen muss.

== Fail: ==
HTTP-Code: 404

== Aufruf, wenn nicht vorher eingeloggt: ==
HPPT-Code 401 Unauthorized
